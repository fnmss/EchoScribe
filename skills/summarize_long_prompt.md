# 长视频访谈总结 Prompt 模板

此文件定义了长视频访谈（播客、深度对话等）的总结指令。修改此文件可以调整 AI 总结的风格和内容。

## Prompt 内容

请对以下长视频访谈转写内容进行详细总结。这是一段较长的对话或访谈，需要更深入的分析。

要求：
1. **整体概述**：用 2-3 句话概括整个访谈的核心主题和结论
2. **参与者介绍**：识别访谈中的参与者（如有），简要介绍其身份
3. **按话题分段总结**：将内容按话题或讨论主题分段，每段包含：
   - 话题标题
   - 核心观点和论述
   - 关键案例、故事或数据
4. **金句摘录**：提取 3-5 句有启发性的原话（用引号标注）
5. **关键要点**：用列表形式总结最重要的 5-10 个要点
6. **使用中文回答**
7. **使用多级标题（# ## ### ####）组织内容**，便于生成思维导图

转写内容：
{text}

## 使用说明

- `{text}` 会被替换为实际的转写文本（最长 80000 字符）
- 修改上方的 Prompt 内容即可改变 AI 总结的风格
- 支持 Markdown 格式
