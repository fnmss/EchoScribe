# AI 总结 Prompt 模板

此文件定义了发送给大模型的总结指令。修改此文件可以调整 AI 总结的风格和内容。

## Prompt 内容

请对以下音视频转写内容进行总结。要求：
1. 提炼核心主题和关键信息
2. 按时间线或逻辑结构分段总结
3. 保留重要的数据、人名、专有名词
4. 使用中文回答
5. 使用多级标题（# ## ### ####）组织内容，便于生成思维导图

转写内容：
{text}

## 使用说明

- `{text}` 会被替换为实际的转写文本（最长 80000 字符）
- 修改上方的 Prompt 内容即可改变 AI 总结的风格
- 支持 Markdown 格式
